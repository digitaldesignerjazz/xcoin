# XCoin – QCoin Integration

XCoin serves as the base cryptocurrency layer that supports and interacts with the QCoin rune economy.

## Key Integration Points
- Liquidity and value transfer between XCoin and runes (especially NovaRune)
- Shared economic incentives through Nexus
- Rune-specific utilities built on top of XCoin settlement
- Potential for rune-backed assets or wrapped representations on the XCoin chain

## Development Focus
- Bridge and liquidity mechanisms
- Cross-layer oracle and data sharing via Nexus
- Aligned governance and parameter adjustment
