# XCoin – NovaRune Integration

NovaRune is the infrastructure-focused rune that rewards mesh operation. XCoin provides the underlying value layer that supports NovaRune incentives and liquidity.

## Integration Goals
- Enable NovaRune rewards to be settled or bridged into XCoin
- Allow XCoin holders to participate in infrastructure staking indirectly
- Shared Nexus-driven economic signals between layers

## Focus Areas
- Bridge mechanics between NovaRune and XCoin
- Cross-layer reward distribution
- Aligned incentives for mesh node operators
