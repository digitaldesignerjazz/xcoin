# XCoin Tokenomics & Monetary Policy

## Overview
XCoin is designed as the foundational cryptocurrency layer with a focus on long-term sovereignty, adaptability, and hybrid value (monetary + meaning-driven).

## Key Parameters (Initial)
- **Ticker**: XCOIN
- **Initial Supply**: To be defined in genesis / launch parameters
- **Inflation Model**: Low or zero base inflation, with dynamic adjustments via Nexus
- **Utility**: Gas fees, staking, settlement, liquidity provision, cross-layer coordination

## Monetary Policy Principles
- **Self-Improving Economics**: Nexus can propose and (with governance) adjust parameters such as inflation rate, staking rewards, and fee structures based on network health and ecosystem performance.
- **Hybrid Value Model**: Supports both traditional monetary use cases and creative/ritual economic activity through integration with the QCoin rune layer.
- **Sovereignty First**: Designed for mesh-native operation with strong privacy and censorship resistance.

## Supply Model
Detailed supply schedule, allocation, and vesting will be defined in coordination with the NovaCoin genesis and overall ecosystem launch strategy.

## Dynamic Adjustment
Nexus monitors key metrics (mesh performance, agent productivity, prototype progress, rune economy health) and can recommend or enact parameter changes to keep the economy healthy and aligned with long-term vision.
